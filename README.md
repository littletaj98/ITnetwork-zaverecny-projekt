# ITnetwork závěřečný projekt

## Popis
Pracovní název projektu (EFIL pojištění) obsahuje minimální požadavky pro splnění zadání.

1. Vytvoření pojištěnce
2. Úprava pojištěnce
3. Smazání pojištěnce
4. Výpis pojištěnců

## Spuštění v localním prostředí

1. npm i
2. npm start

## Dev stack
1. Node
2. Express
